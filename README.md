# 🎤 Speech-to-Text Converter

## Overview

Speech-to-Text Converter is a MERN Stack application that converts audio files into text using the Deepgram API. Users can upload or record audio, generate transcripts, and save them securely in MongoDB.

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB Atlas
* Tailwind CSS
* JWT Authentication
* Deepgram API

## Features

* User Signup & Login
* Audio Upload & Recording
* Speech-to-Text Conversion
* Transcript History
* Search Transcripts
* Download as PDF/TXT
* Analytics Dashboard
* Dark Mode
